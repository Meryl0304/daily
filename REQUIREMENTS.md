## 二、首页（日历 ①）

### 2.1 极简模块

#### 日期
- 字体加大（18px → 20px+）
- 增加**农历日期**（方案 A：仅农历月日 + 生肖年，不接外部 API）
- 农历以较小灰度字体呈现，适配夜间模式

#### 星标任务（每日最重要）
- 以**半透明圆角方框**为底
- 未完成时底色**暗红**（`rgba(200,50,50,0.08)`）
- 完成时底色**绿色**（各主题不同绿色色值）
- **点击跳转日详情页**，锚定到该任务（背景短暂高亮闪烁）
- 日详情勾选完成后实时反映到首页颜色

#### 日常提醒
- 仅显示**当前日期该出现的 habit**
- 过滤逻辑：每日 → 天天显示；每周 → 匹配星期几；每月 → 匹配日期
- 保持进度百分比和打卡按钮

#### 喝水
- 六点圆圈 → **一条进度条**
- 填充色：蓝色（`#5b9bd5` 左右）
- 满杯 6/6 时旁边浮现提示「皮肤好好，喝水饱饱 🌟」，几秒后渐消
- `[−]` 和 `[+]` 按钮在进度条两端，`[+]` 做大一点方便按
- 进度条本身不可拖拽，只通过按钮增减

#### 🌸 花开（原「许愿瓶」）
- 图标：牡丹 SVG（风格 A·简约五瓣），**无文字标题**
- 内容：瓶子 + 幸运星散落其中
- 点击星星：弹出查看详情 + 「摘掉这颗星」
- 「+ 存一颗」按钮
- 近 30 天计数

#### 今日显化
- 每天从四个维度各随机抽一句：

| 维度 | 说明 |
|------|------|
| 💼 事业 | 钱多事少离家近、有所成就 |
| 💰 金钱 | 财务自由、被动收入 |
| 🦋 自由 | 松弛生活、幸运降临 |
| 🌱 自我 | 自信、底气、26岁女性力量 |

- 每维度 8-10 句，总计 ~40 句
- 不支持用户自定义（预设库循环）

#### 配置项
极简模块显示项（设置页可勾选）：
```
日期 ｜ 今日最重要 ｜ 日常提醒 ｜ 喝水 ｜ 🌸花开 ｜ 今日显化
```
- 移除「营养品」
- 「今日事件」为独立区域，不在极简模块内，不受此配置控制

### 2.2 日历格子

- 年月字体加大，旁边加**月份快速选择器**（点击弹出面板）
- 「今天」按钮改为圆角按钮样式
- 每格信息层级（字号递减）：

```
┌──────┐
│  18  │  ← 日期数字 13px（主体）
│ 3/5  │  ← 完成/总数 10px（次要，灰色）
│ ●●◉  │  ← 标记点 4px
└──────┘
```

| 标记 | 含义 |
|------|------|
| ● 绿 | 当天有至少一项已完成任务 |
| ● 蓝 | 当天有日程事件 |
| ◉ 红 | 当天有逾期未完成的任务 |

- **点击日期 → 高亮选中 → 极简模块切换到该日数据（不跳转）**
- 双击日期或点击小箭头 → 进入日详情页

### 2.3 逾期任务区
- 逾期任务旁增加「接入日程」按钮
- 点击弹出迁移确认框：

```
┌──────────────────────────────────────────┐
│ 将「写月度报告」接入日程？                 │
│                                          │
│                                          │
│                                          │
│ 今天（6/18）可用 2h                       │
│   ├─ ✅ 整理大纲  60min                  │
│   └─ ✅ 填写数据  40min                  │
│                                          │
│ 明天（6/19）可用 3.5h                     │
│   └─ ✅ 校对      30min                  │
│                                          │
│ ⛔ 后天（6/20）是休息日  [取消休息]         │
│                                          │
│ ↳ 预计 6月19日 全部完成                    │
│                                          │
│       [确认分配]        [取消]             │
└──────────────────────────────────────────┘
```

**贪心算法**：子任务从大到小排序，优先塞进最早有剩余容量的天，塞不下则往后推。
**休息日**：默认跳过不分配，用户可点 `[取消休息]` 将其加入分配池（按钮变为 `[恢复休息]` 可反悔），系统重跑贪心算法。
**无子任务时**：先评估当天能否容下该任务；容不下则建议拆分。
每次只处理一个任务，不做批量。

### 2.4 今日事件

- 位置：日历格子下方，与极简模块独立
- 内容：当天 `type: 'event'` 的提醒条目
- 显示格式：时间 + 事件名称 + ✓/⚠ 确认状态
- 全天事件排最后，按时间升序
- 最多显示 3 条，超出显示「+N 条」（点击展开）
- 今天无事件时显示「今天没有安排 🌿」
- **点击条目 → 弹出小窗**：

```
┌─────────────────────────┐
│  团队周会                │
│  今天 09:00              │
│  提前 15 分钟提醒         │
│                         │
│  状态: ⚠ 尚未确认        │
│                         │
│  [确认完成]  [关闭]      │
└─────────────────────────┘
```

- 小窗内可直接确认，与日详情日程确认双向同步
- 过往日期的事件在日详情页查看

## 三、日详情页（②）

### 3.1 编辑模式
- **移除页眉 ✏️ 按钮**，编辑入口仅保留在日详情页内
- 每次进入日详情页，编辑模式**默认关闭**
- 切换日期 → 自动退出编辑模式
- 离开日详情页时如处于编辑模式 → 弹提示「退出编辑模式？」
  - 「离开」定义：切换到其他 tab、或点击返回按钮回到首页
  - 仅切换日期不算离开，仍在日详情页内

### 3.2 标题与导航
- 顶部：**← 前一天 | 日期选择器 | 后一天 →**
- 移除底部积分展示
- 过往日期标题：浅琥珀底色 + 「📋 过往」标签

### 3.3 任务列表
- 标题「日常任务」→ **「今日事项」**
- 标题字体加大加粗 + 主题色，与内容行形成层级落差
- 新增任务时可选填**预计用时**（分钟），不强制
- 新增任务时可选填**分类标签**

### 3.4 子步骤折叠
- ≤ 2 个：全部展开
- > 2 个：前 2 个展开，其余折叠，显示「展开全部（还有 N 项）」
- 展开后显示「收起」

### 3.5 任务行交互
- 每项任务行显示「▶」（播放键）启动凝神
- 母任务：计时器持续跑，子步骤列表在底部，手动勾选切换
- 子任务/单一任务：直接计时

### 3.6 过往日期
- 标题栏琥珀暖色底色
- 「📋 过往」标签
- 任务圆圈浅色边框（不实心），不可点击
- 已逾期的未完成任务旁出现「接入日程」按钮

### 3.7 日程事件
- 条目增加**确认勾选**（✓ / ○）
- 和通知系统双向同步
- 新增日程时可设置重复提醒间隔

## 四、凝神 🎯（原「冲刺」）

### 4.1 命名与入口
- 名称：**凝神**
- 底部导航独立 Tab
- 任务行「▶」点击 → 进入凝神页并关联该任务
- 空闲态（未在计时）：显示统计面板（热力图/趋势/时段），顶部「自由凝神」入口 + 今日待凝神任务列表
- 自由模式入口同样在空闲态可见

### 4.2 计时器 UI
- 方向 **B · 暮色温润**
- 暖金色渐变计时环 + 微弱光晕
- 专注中背景略微变暗
- 子步骤以**气泡框**展示在计时环下方，点击划掉

### 4.3 计时逻辑
- 默认 25 分钟番茄钟（可在设置调整）
- **连续型**计时：子步骤之间不停表
- 休息按钮 ⏸ → 5 分钟倒计时（舒缓绿色环）
- 休息结束不自动返回，等用户点「回来了」
- `actual_minutes = 总时间 - Σ休息时间`

### 4.4 按钮
| 状态 | 按钮 | 说明 |
|------|------|------|
| 启动前 | ▶ 开始 | 红色播放键，点击开始计时 |
| 计时中 | ⏸ 休息 | 进入 5 分钟休息 |
| 休息中 | 回来了 | 回到正常计时 |
| 计时中 | ✅ 完成 | 弹出随机夸赞，记录用时 |
| 计时中 | ⏹ 停止 | 进入分流引导（非文字，只用符号） |

### 4.5 夸赞语（完成时随机抽取）
- 「太棒啦！又完成一项！」
- 「你真的很厉害诶～」
- 「一步一步来，超棒的！」
- 「辛苦了！这份专注很值得」
- 「好耶！给自己鼓个掌～」

### 4.6 停止分流

**点的是子任务 →**

```
┌──────────────────────────────┐
│  💭 刚才的进度如何？           │
│                              │
│  已完成的: [___________]      │
│            [+ 添加]          │
│  还没碰的: [___________]      │
│                              │
│  💡 或许可以拆成更小的步骤     │
│  [保存拆分，继续凝神] [结束]   │
└──────────────────────────────┘
```

**点的是单一任务 →** 同上，引导更积极。保存后已完成的变成子任务标记完成，未完成的变成新子任务。

**点的是母任务 →** 显示所有子步骤状态面板，勾选完成/进行中/未碰。

### 4.7 自由模式
- 保留凝神页顶部入口「自由凝神」
- 不绑定任务直接开始
- 记录 `task_id: null`

### 4.8 统计面板
- Tab 切换：`[热力图] [趋势] [时段]`
- **热力图**：GitHub 风格，横轴星期、纵轴周次
- **趋势图**：柱状图，日/周/月切换，可按类别堆叠
- **时段图**：06:00-24:00 折线，显示高频时段
- 凝神进行中时统计区域可折叠

### 4.9 数据结构
```javascript
// 冲刺记录
sprintLog: [{
  date, start_time, duration_min,
  task_id: null,        // 关联任务（null=自由模式）
  task_category: null,   // 任务分类
  actual_minutes: 0,     // 实际用时
  completed: true,       // 正常完成 or 提前退出
  notes: ''              // 用户备注
}]

// 任务新字段
tasks: [{
  ...现有字段,
  est_minutes: null,       // 预估用时（分钟），可选
  category: null,          // 分类标签
  completed_date: null,    // 完成日期（哪天打勾）
  total_focus_minutes: 0   // 累计专注时间
}]
```

## 六、拾语 🌐（原「语言角」）

### 6.1 命名
- 名称：**拾语**
- 底部导航第五个 Tab

### 6.2 语言切换
- Tab 切换：`[English] [日本語] [Français]`
- 每种语言独立配色（EN 暖棕 / JP 淡紫 / FR 淡蓝）

### 6.3 语句气泡
- 页面中部随机漂浮排列，上下微浮动动画
- 每次显示 5 个气泡，从 30 句库中随机抽取
- 点击「下一句」后该气泡替换为新的随机句
- 下拉刷新可全部换一批
- 点击气泡 → 弹出详情：
  - 原文 + 读音 + 中文翻译 + 语法标注
  - 「收藏到生词本」「下一句」
- 长按 / 双击收藏

### 6.4 数据
- 每种语言 **30 句**，预设于 `sentences_data.js`
- 每句带：`text`、`reading`、`translation`、`grammar`

### 6.5 生词本
- 底部列表，显示所有收藏的句子
- 按语言 tag 分类（EN/JP/FR 彩色标签）
- 手动添加：语言选择 + 原文 + 释义 → 存入

### 6.6 碎片复习
- 闪卡模式：正面原文 → 点击翻面 → 背面释义
- 「还不太熟」「记住了 ✓」按钮
- 记住的降低出现频率

## 七、提醒与通知

### 7.1 提醒引擎
- 30 秒轮询检查（不变）
- 通知横幅**不自动消失**，停留直到用户操作
- 若用户无操作，每 **3 分钟**重复提醒，最多 **5 次**
- 声音从单频 → **双音短提示**（1-2 秒）

### 7.2 通知操作
- 「知道了」→ 对应 event 标记为已确认，通知记录显示 ✓
- 「延后 5 分」→ 5 分钟后再次弹出

### 7.3 铃铛红点
- 未确认通知数显示在页眉 🔔 红点
- 全部确认后红点消失

### 7.4 日详情日程确认
- 每条日程事件前 ✓ / ○ 勾选
- 勾选/取消 → 和通知系统双向同步

## 八、智能调度（三层）

> 数据管道本轮全部铺设，功能逐层激活

### 8.1 层次一 · 任务负载感知（本轮可用）
- 今日所有任务累计 `est_minutes`
- 用户每日容量 `daily_capacity`（初始手动设 6~8h，随数据自动调整）
- 逾期迁移时计算剩余容量 → 建议截止天数
- 逻辑示例：今日剩余 50 分 → 子任务 C(40分) 塞入；明日剩余 90 分 → A(60)+B(30) 塞入

### 8.2 层次二 · 个人节奏学习（需要 2-4 周数据）
- 凝神记录累积 → 统计分析：
  - 按任务类别平均用时
  - 高频时段分布
  - 完成率趋势
- 异常提示：「这项用时超出同类均值，是否需要备注？」

### 8.3 层次三 · 智能子任务插入（需要层次二数据基础）
- 逾期任务迁移时自动拆分建议
- 无历史数据时降级：按手动 `est_minutes` + 容量计算
- 数据充足后：按同类任务历史 + 用户高频时段 + 当日负载综合分配
- 休息日默认跳过，弹确认询问
